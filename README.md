# 💰 تطبيق مدير الحسابات والمصاريف - Finance Manager

تطبيق Flutter احترافي لإدارة المصاريف والحسابات المالية بنظام القيد المزدوج (Double Entry Accounting).

## 🌟 الميزات الرئيسية

### للمستخدم العادي:
- ✅ إضافة مصروف/دخل بسرعة فائقة
- ✅ تصنيفات جاهزة (طعام، نقل، إيجار، إلخ)
- ✅ إدارة حسابات متعددة (كاش، بنك، بطاقات)
- ✅ رسوم بيانية تفاعلية
- ✅ تقارير PDF قابلة للمشاركة
- ✅ دعم كامل للغة العربية
- ✅ وضع ليلي (Dark Mode)

### للمحترفين والأعمال:
- ✅ نظام محاسبي Double Entry مخفي
- ✅ دفتر الأستاذ (General Ledger)
- ✅ إدارة الديون (لي / علي)
- ✅ تتبع العملاء والموردين
- ✅ كشوف حساب تفصيلية
- ✅ دعم تعدد العملات

### الأمان والربح:
- ✅ قفل بالبصمة/الوجه
- ✅ نسخ احتياطي سحابي
- ✅ إعلانات AdMob
- ✅ اشتراكات Premium

## 📋 المتطلبات المسبقة

قبل البدء، تأكد من تثبيت:

1. **Flutter SDK** (3.24.0 أو أحدث)
2. **Android Studio** أو **VS Code**
3. **Firebase CLI** (اختياري)

---

## 🚀 التثبيت والتشغيل

```bash
# 1. استنساخ المشروع
git clone <repository-url>
cd expense_manager

# 2. تثبيت المكتبات
flutter pub get

# 3. إنشاء ملفات التوليد
flutter pub run build_runner build --delete-conflicting-outputs

# 4. تشغيل التطبيق
flutter run
```

## 📁 هيكل المشروع

```
lib/
├── core/           # الطبقة الأساسية (خدمات، ثيمات، أدوات)
├── data/           # طبقة البيانات (قواعد، نماذج، مستودعات)
├── domain/         # طبقة الأعمال (كيانات، حالات استخدام)
├── presentation/   # طبقة العرض (شاشات، widgets، providers)
└── main.dart       # نقطة الدخول

firebase/          # إعدادات Firebase
assets/            # الصور، الخطوط، الأيقونات
```
